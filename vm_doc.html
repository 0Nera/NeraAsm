Инструкции

ST - начало программы(необ)
ED - Конец программы
		>>>Ячейки<<<
На данный момент присутствует 2 ячейки для хранения данных:
	C - Символьная ячейка
	I - Целочисленная
		>>>Параметры ячеек<<<
	W - запись числа (от 00 до 99) в ячейку
	P - вывести данные из ячейки
	+ - увеличить число в ячейке
	- - уменьшить число в ячейке
	<имя_ячейки> - присвоить ячейке значение другой ячейки

Примеры:

Hi!

ST CW 72 CP CW 73 CP CW 33 CP ED

XD

ST IW 88 CX CP CP CW 67 CP ED
